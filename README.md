El juego del ahorcado consiste en adivinar una palabra secreta antes de que se complete un dibujo del "ahorcado".

Preparación: Se elige una palabra y se representan sus letras con líneas.
Adivinar letras: Los jugadores proponen letras. Si están en la palabra, se colocan en las líneas; si no, se dibuja una parte del "ahorcado".
Fin del juego: Ganas si completas la palabra antes de terminar el dibujo; pierdes si el dibujo se completa primero.
